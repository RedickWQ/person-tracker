# Person Tracker UI 重构设计方案

## Context

用户希望重构 person-tracker 项目的 UI，使其更加美观、时尚。选定方案为 **Notion 风渐进重构**，重点优化卡片设计、颜色系统和动画交互。

## 设计原则

1. **柔和圆角** — 卡片圆角 20px，按钮 12px
2. **轻盈投影** — 多层柔和投影，hover 时加深
3. **渐变背景** — 四种目标类型各有独特渐变色
4. **流畅动画** — 所有交互有过渡动画

## 颜色系统

|---------|--------|----------|
| 副业 💼 | #8B5CF6 紫 | #F3E8FF → #E9D5FF |
| 思维训练 🧠 | #06B6D4 青 | #E0F2FE → #BAE6FD |

## 卡片设计规范

### 当前状态
- 卡片圆角：16px
- 背景：单色或简单渐变
- 投影：轻微单层投影
- Hover：基本无动效

### 目标状态
- 卡片圆角：20px
- 背景：根据目标类型使用独特渐变
- 投影：多层柔和投影 (0 4px 6px rgba)
- Hover：Y轴上移 4px + 阴影加深 + 过渡 200ms

## 动画规范

| 元素 | 动画类型 | 时长 |
|------|----------|------|
| 页面切换 | 淡入淡出 | 300ms |
| 按钮 hover | 缩放 1.02 + 阴影加深 | 200ms |
| 卡片 hover | Y轴上移 4px + 阴影扩散 | 200ms |
| 加载状态 | 脉冲动画或骨架屏 | - |

## 实施任务

### 任务 1：全局样式更新
- 更新 `src/index.css` 全局样式
- 添加 CSS 变量定义主题色
- 设置全局动画过渡

### 任务 2：Dashboard 卡片重构
- 更新 `src/pages/Dashboard.css` 卡片样式
- 应用新圆角、投影、渐变
- 添加 hover 动画效果

### 任务 3：GoalDetailPage 样式优化
- 更新 `src/pages/GoalDetailPage.css`
- 统一卡片风格
- 优化阅读计时器样式

### 任务 4：其他页面适配
- 检查并更新其他页面的卡片样式
- 确保视觉一致性

## 验证标准

1. 四种目标类型有独特且区分度高的颜色
2. 卡片 hover 有流畅的上浮动效
3. 整体风格接近 Notion 的清爽柔和感
4. 动画流畅，无明显卡顿
